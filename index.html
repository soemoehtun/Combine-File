How to use
1
Import files. Drag and drop your Excel or CSV files or click browse files to select them.
2
Set start row. Choose Row 1 to keep headers, Row 2 to skip the heading, or Custom Row for a specific starting row.
3
Select sheet. Pick the sheet to combine from the dropdown. Available sheets from all imported files are listed.
4
Review headers. Green status means all headers match. Grey shows mismatches — toggle files or change sheet to re-check.
Row 1 (Include Heading) — Each file's heading row is included in the output
Row 2 (Skip Heading) — Heading row is taken from the first file only; subsequent files skip row 1
Custom Row (3) — Skips first 2 rows of each file (header + 1st data row)
